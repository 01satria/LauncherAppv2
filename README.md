# Satria Launcher — Kotlin + Jetpack Compose

Migrasi penuh dari React Native ke Kotlin native.


## Catatan
- Internet permission sudah ada via `INTERNET` yang di-grant otomatis (normal permission)
- Avatar disimpan di `filesDir` internal storage
- Semua data (todo, countdown, weather locations) persist via DataStore
